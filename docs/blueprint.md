# **App Name**: StaysKenya

## Core Features:

- Accommodation Listings: Display available accommodations with details, images, and pricing.
- Search and Filtering: Enable users to search and filter accommodations based on location, price, amenities, and dates.
- Booking Management: Allow users to book accommodations, manage bookings, and receive confirmations.
- User Reviews and Ratings: Incorporate reviews and ratings for each accommodation to provide user feedback.
- AI-Powered Recommendations: AI tool recommends relevant accommodations based on user preferences, booking history, and search criteria.
- Homepage: Homepage with headline: “Find Airbnbs & Guest Rooms in Kenya”
- Listings Page: Listings page with Images, Price per night, Area/location name, Category (Budget, Mid-range, Luxury), Call and WhatsApp buttons
- Post Your Airbnb Page: “Post Your Airbnb” page for hosts to submit listings
- About / Trust Page: About / Trust page explaining the local and national focus
- Firebase Authentication: Firebase Authentication for hosts
- Firestore Database: Firestore database for listings
- Firebase Storage: Firebase Storage for images
- Admin Approval: Admin approval before listings go public

## Style Guidelines:

- Primary color: Warm beige (#F5F5DC), evoking feelings of earthiness and tranquility to represent Kenya's landscape.
- Background color: Off-white (#FAFAFA), providing a clean and modern backdrop.
- Accent color: Muted green (#A9BC8A) for highlighting key elements.
- Headline font: 'Playfair', a modern serif with an elegant, high-end feel. Body font: 'PT Sans' a humanist sans-serif
- Use minimalist icons related to accommodation types, amenities, and locations.
- Mobile-first, responsive design ensuring a seamless experience across devices.
- Subtle transitions and animations to enhance user interactions and provide visual feedback.
